unit -- a measurement of length equal to 100 centimeters or approximately 39.37 inches.
pixel -- a unit of measurement for digital images, representing a single point in a graphic image.
rem -- a unit of measurement in CSS that is relative to the font size of the root element (usually the element). rem stands for "root em" and is used to create scalable and responsive designs in web development. One rem is equal to the font size of the root element, which allows for consistent sizing across different devices and screen sizes.
em -- a unit of measurement in CSS that is relative to the font size of the current element.
